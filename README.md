# OWASP Juice Shop Penetration Testing

This project demonstrates vulnerability assessment and penetration testing performed on the OWASP Juice Shop application.

## Objective
To identify security vulnerabilities in a deliberately insecure web application.

## Tools Used
- Kali Linux
- Burp Suite
- Browser DevTools
- OWASP Juice Shop

## Vulnerabilities Identified
- Broken Authentication
- XSS (Cross Site Scripting)
- Admin Login Bypass
- Sensitive Data Exposure
- Insecure Direct Object Reference

## Skills Demonstrated
- Web Application Pentesting
- Vulnerability Analysis
- Exploit Demonstration
- Security Reporting
